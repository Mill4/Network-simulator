Detailed Description

class Simulator

Description:

The Simulator is a class that stores pointers of all the network elements in different list containers. The class allows basic manipulation of these network elements through the functions described below.

The class obviously has a lists of all nework elements pointers as a private member. This class is not copyable (also means not assignable), so copy and assignment operations for this class is prevented.
