Detailed Description

Packet class represents a single network packet that can be sent between nodes.

Packet is originally split from a bigger unit of data. In our system we use Ethernet MTU as a default max packet sixe (1500 bytes)

The packet header has an ID (4 letter string - 4 bytes)
